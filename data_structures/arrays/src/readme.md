# Some functions to work with arrays and others data structures


* [array\_struct.c](array_struct.c) core that define a ArrayLike struct.
* [binary\_search.c](binary_search.c) perfom a binary search in the ArrayLike struct.
* [cocktailSort.c](cocktailSort.c) perform a cocktail sort in the ArrayLike struct.
* [randPopArray.c](randPopArray.c) populate the ArrayLike struct with random number.
* [sortArray.c](sortArray.c) perform a quickSort in the ArrayLike struct.
